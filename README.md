# Customer Segmentation Analytics using RFM Modeling (Python)

## 📌 Project Overview
This project addresses a classic database marketing problem: dividing a customer base into distinct groups to prevent churn and maximize promotional campaign return on investment (ROI). Utilizing Python Pandas, I built an analytical pipeline to evaluate consumer behavior across Recency, Frequency, and Monetary parameters.

## 🛠️ Tech Stack & Tools
- **Language:** Python 3
- **Libraries:** Pandas, NumPy, Datetime
- **Methodology:** RFM Customer Segmentation Matrix

## 📊 Analytical Insights & Strategic Impact
- **Champions (Core Loyalists):** Identified segments with high monetary values and near-zero recency lag (e.g., Customer 501 & 503). *Strategy:* Target with early-access product drops, do not dilute with high discount codes.
- **At-Risk Segments:** Isolated high-value accounts that have completely dropped off activity windows (e.g., Customer 505 spent $500 but hasn't interacted in 149 days). *Strategy:* Deploy automated "win-back" email sequences instantly.
